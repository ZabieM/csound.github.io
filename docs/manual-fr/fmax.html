fmax

fmax — Function valeur maximale.

Description

Retourne le maximum de ses deux arguments.

Syntaxe

ires[] fmax iarg1[], iarg2[] 
kres[] fmax karg1[], karg2[]
ires[] fmax iarg1[], iarg2 
kres[] fmax karg[], karg2

Initialisation

iarg[]1/2, iarg2 -- les opérandes.

Exécution

karg[]1/2, karg2 -- les opérandes.

Voici un exemple de l'opcode fmax. Il utilise le fichier fmax.csd.

Exemple 346. Exemple de l'opcode fmax.

 print iRes[ik]
 ik += 1
od
